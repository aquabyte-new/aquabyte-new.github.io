README

This README would normally document whatever steps are necessary to get your application up and running.

First time you clone:

Make sure you have SSH key linked to your github account!

git clone --recursive git@github.com:aquabyte-new/aquabyte_marketing_frontend.git git submodule update --init --recursive

How do I get set up?

install http://harpjs.com/

then run the server by "harp server"

Use this code in case you get stuck

sudo npm install --unsafe-perm --verbose -g harp

Deployment

Run git submodule update --init --recursive

Connect to heroku

  • install heroku CLI and login
  • cd to www
  • heroku git:remote -a aquabyte-marketing-frontend
  • git pull heroku master

Run harp compile then it'll generate a www folder Upload files in the www folder to server by:

  1. cd to www
  2. git add . and git commit -m "message here" with a meaningful summary of changes
  3. git push heroku master

You might have to do git pull heroku master to grab lastest remote changes before deploy
